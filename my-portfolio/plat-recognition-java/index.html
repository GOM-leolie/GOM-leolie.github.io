Comvis Project: Plate Number Recognition (Indonesia)

Plate number recognition (Indonesian plate) using image manipulation technique to extract the number plate from the given photo.
Project period: May 2015 - July 2015.

Background

This project was created in a collaboration with a student from University in Indonesia as part of his final paper. The goal was to take a raw image and perform a string of image manipulation and processing technique to extract the features of the image to produce the output, in this case it is a string containing the extracted plate number.

This project assumes that the given image is clear, straight, and in landscape orientation. The program will skip the image pre-processing technique. This project is tested with the Indonesian's number plate only.

It was developed with Java as the programming language with Javax.Swing UI framework.

Process

The program will load the image from the user (png format), read it into a stream, and extract the pixel and split it into RGB colour to be manipulated. The user then be able to try and perform various of image manipulation technique, before proceeding with the image processing technique to extract the features.

Image Manipulation

Greyscale

Greyscale is the first image manipulation technique to be performed before applying other technique. The goal of this process is to reduce the noise of the image, which allows us to focus on object that we want to extract.

To transform an image to a Greyscale, the colour of each pixel need to be manipulated. The formula is :
(R x ) (G x ) (B x )
As seen on the picture, the greyscale image tone down the colour and categorised into black and white group. The greyscaled image then can be filtered further by using couple of manipulation technique to smoothen the edge.

Mean and Median Filter

After the image has been greyscaled, sometimes we can move directly to edge detection technique, if the image does not have a lot of background noise. However, if the image is busy, further filtering needs to be performed to reduce these noises. Mean and median filter are two of the popular techniques to filter the background noise.

As the name sounds, mean filtering means finding the average value from the 8-neigboring edge of the pixel to replace the value of the pixel. On the other hand, median filtering means finding the median value of the neighboring pixels instead of the mean value.

Sometimes median filter works better than the mean filter, and vice versa, depending on the image lighting, colour, and other attributes. Sometimes both of the filters need to be applied together to further remove the noise.

As seen on the picutre, mean filter looks blurier, while the median filter looks sharper.
